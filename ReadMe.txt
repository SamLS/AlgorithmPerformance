Select a mode with input of 1 or 2.

The two modes are:
1- Single Algorithm Performance-
		Select an algorithm with input 1 or 2.
		The two algorithms are:
			1- First Come First Serve
			2- Round Robin
				a. For Round Robin, user must enter a quantum number as an integer.
			2- Algorithm Comparing	

User must enter a quantum number as an integer for Round Robin.

Data is generated and printed to the console, as well as written to a file named “Performance”. This file is saved to the directory in which the program is also placed.
